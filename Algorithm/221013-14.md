## Sort &rarr; 순서 바꾸기

1. Bubble Sort      &rarr; O(n<sup>2</sup>)
2. Insertion Sort   &rarr; O(n<sup>2</sup>)
3. Selection Sort   &rarr; O(n<sup>2</sup>)
4. Quick Sort       &rarr; O(nlogn)
5. Merge Sort       &rarr; O(nlogn)
